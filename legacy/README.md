# Legacy Archive

This directory holds deprecated tools and exploratory files that are no longer
part of the supported application path.

Current supported path:

- `src/fba/app.py`
- `src/fba/auth.py`
- `src/fba/yahoo_api.py`
- `frontend/`

Archived here:

- older manual helpers
- exploratory Yahoo API scripts
- deprecated credential examples tied to removed flows

Nothing in this directory is required to run the current app.
